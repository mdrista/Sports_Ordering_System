The Sports Ordering System is a text-based program designed for staff and telemarketers to quickly and accurately create customer orders for sports equipment. Products are organised into three categories: Shoes, Clothing, and Accessories, each with three items. Users navigate a numbered menu to select products, enter quantities, and view the total cost of their order. Input validation ensures the program handles invalid entries without crashing, and a clear order summary is displayed at the end. The system is user-friendly, reliable, and flexible, making it easy to add new products or categories in the future.
